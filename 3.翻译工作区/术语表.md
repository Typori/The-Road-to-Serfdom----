# 《通往奴役之路》全书术语表

## 政治制度与意识形态

| 英文 | 中文 | 说明 |
|------|------|------|
| liberalism | 自由主义 | 强调个人自由、法治、市场经济的政治哲学 |
| liberal | 自由主义者 | 信奉自由主义的人 |
| socialism | 社会主义 | 主张生产资料公有、计划经济的政治制度 |
| socialist | 社会主义者 | 信奉社会主义的人 |
| collectivism | 集体主义 | 强调集体利益优于个人利益的思想体系 |
| individualism | 个人主义 | 强调个人权利和自由的思想体系 |
| democracy | 民主 | 人民主权、多数决策的政治制度 |
| totalitarianism | 极权主义 | 国家权力无限扩张、控制社会各方面的制度 |
| fascism | 法西斯主义 | 极权主义的一种形式，强调民族主义和独裁统治 |
| Nazism | 纳粹主义 | 德国法西斯主义，强调种族优越性 |
| authoritarianism | 威权主义 | 权力集中于少数人或一个人的制度 |
| dictatorship | 独裁统治 | 由独裁者掌握绝对权力的制度 |
| capitalism | 资本主义 | 以私人资本和市场为基础的经济制度 |

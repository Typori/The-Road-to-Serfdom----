# 翻译工程工具集

本目录包含用于辅助《通往奴役之路》翻译工程的各类工具。

## 工具列表

### 1. split_chapters.py - 原文章节拆分工具

**功能**：根据原文中的目录结构，自动将完整原文拆分为多个独立的章节文件。

**使用方法**：

```bash
python split_chapters.py
```

**输出**：
- 在 `2.原文资料/chapters/` 目录下生成16个章节文件
- 生成 `00_章节索引.md` 索引文件

**特点**：
- ✅ 自动识别16个章节（One - Sixteen）
- ✅ 处理特殊格式（如第四章标题中的Unicode引号）
- ✅ 处理跨行标题（如第十五章）
- ✅ 使用单词边界精确匹配，避免误匹配（如FOUR vs FOURTEEN）
- ✅ 生成清晰的章节索引文件
- ✅ 文件名自动处理特殊字符

**技术要点**：
1. **引号处理**：第四章标题使用Unicode左右引号（U+201C/U+201D），而非ASCII引号
2. **换行处理**：第十五章标题在原文中跨越两行
3. **精确匹配**：使用 `\b` 单词边界确保 `FOUR` 不会匹配到 `FOURTEEN`

**配置**：
工具使用默认路径，可在 `main()` 函数中修改：
- `SOURCE_FILE`: 原文文件路径（默认：`2.原文资料/The-Road-to-Serfdom.txt`）
- `OUTPUT_DIR`: 输出目录（默认：`2.原文资料/chapters`）

**功能**：将对照译文文件转换为纯译文格式，方便阅读。

**使用方法**：
